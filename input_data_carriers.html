Data common for all types#

Data specific for each type#

el#

parameter

type

description

powerflow_method

str

“transport” or “dc_pf”

reference_node

str

id of node used as reference for voltage angles (with dc_pf method)

el_reserve_margin

float

required globally spinning reserve (MW).

el_backup_margin

float

(optional) required backup margin in case of genertor failure (MW)

reserve_storage_minutes

int

how long (minutes) stored energy must be sustained to count as reserve

No additional data required
